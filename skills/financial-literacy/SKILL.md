# 财商训练工具顾问

基于《财商训练工具手册》，提供 24 个财商训练工具的检索与应用指导。

## 模块总览

### 第一层：财富认知（4个）
| 模块 | 核心问题 | 适用场景 |
|------|---------|---------|
| 欲望管理 | 这个"想买"是需求还是欲望？ | 区分必要消费与冲动消费 |
| 稀缺心态 | 你是在"缺钱心态"还是"富足心态"下做决策？ | 识别稀缺心态对财务决策的扭曲 |
| 消费心理学 | 你在为"拥有"付费还是为"感觉"付费？ | 理解消费背后的心理驱动 |
| 价值认知 | 一件东西的"价格"和"价值"一样吗？ | 建立对金钱和价值的正确认知 |

### 第二层：储蓄与预算（4个）
| 模块 | 核心问题 | 适用场景 |
|------|---------|---------|
| 记账习惯 | 钱都花到哪了？ | 建立收支意识和消费透明度 |
| 预算管理 | 收入如何合理分配？ | 制定月度/年度预算计划 |
| 延迟满足 | 现在的快乐 vs 未来的安全，怎么选？ | 克服即时满足冲动 |
| 50/30/20法则 | 必需/想要/储蓄，占比合理吗？ | 快速建立预算框架 |

### 第三层：投资理财（4个）
| 模块 | 核心问题 | 适用场景 |
|------|---------|---------|
| 复利思维 | 时间如何放大财富？ | 理解长期投资的力量 |
| 资产配置 | 不要把鸡蛋放在一个篮子里？ | 分散风险、平衡收益 |
| 风险评估 | 你能承受多大损失？ | 匹配投资品种与风险承受力 |
| 指数基金入门 | 为什么普通人应该先考虑指数基金？ | 低门槛、低风险起步投资 |

### 第四层：财务规划（4个）
| 模块 | 核心问题 | 适用场景 |
|------|---------|---------|
| 财务目标设定 | 你的财务目标具体吗？SMART吗？ | 制定可量化、有时限的财务目标 |
| 现金流管理 | 每月流入多少、流出多少、净流多少？ | 掌握资金的流动方向 |
| 应急基金 | 如果突然失业/生病，能撑几个月？ | 建立安全垫，应对突发状况 |
| 财务自由路径 | 被动收入能覆盖支出吗？ | 规划从依赖工资到财务自由的路径 |

### 第五层：保险与保障（2个）
| 模块 | 核心问题 | 适用场景 |
|------|---------|---------|

| 险种选择 | 重疾/意外/寿险/医疗，优先级是什么？ | 合理配置保险方案 |

### 第六层：债务管理（2个）

### 第七层：税务意识（2个）

### 第八层：高级财商（2个）

## 使用流程

### 1. 理解用户场景
判断用户处于什么财务情境：刚开始理财、需要控制消费、考虑投资、规划未来、面临债务、需要保险……

### 2. 检索相关模块
根据 `index.json` 中的关键词和场景标签匹配。常见场景映射：

| 场景 | 推荐模块 |
|------|---------|
| "不知道钱花哪了" | 记账习惯 + 50/30/20法则 + 现金流管理 |
| "想攒钱但总是存不下来" | 延迟满足 + 欲望管理 + 预算管理 |
| "该不该投资/怎么开始" | 风险评估 + 指数基金入门 + 复利思维 + 资产配置 |
| "信用卡/花呗压力" | 好债与坏债 + 信用管理 + 消费心理学 |
| "想买保险但不知道买什么" | 保险意识 + 险种选择 + 应急基金 |
| "怎么实现财务自由" | 财务自由路径 + 被动收入构建 + 复利思维 |
| "刚开始工作/理财" | 价值认知 + 记账习惯 + 50/30/20法则 + 应急基金 |
| "有房有车/中年规划" | 资产配置 + 税收基础 + 财富传承 |

### 3. 加载详细内容
根据 `index.json` 中的条目，读取 `references/` 目录下对应文件的详细内容。

### 4. 辅助分析
用训练工具帮助用户：
- **觉察** 自己的财务习惯和心理模式
- **量化** 收支、储蓄率、投资回报率等关键指标
- **规划** 短期目标和长期路径
- **行动** 给出可立即执行的下一步
- **迭代** 定期复盘，持续优化财务状况

## 原则

- **不替代专业建议**：财商训练是认知提升，不替代持牌理财顾问的专业意见
- **因人而异**：每个人的收入、家庭、风险偏好不同，方案必须个性化
- **先防守后进攻**：先建立应急基金和控制消费，再考虑投资
- **知行合一**：理解概念只是第一步，关键在执行
- **长期主义**：复利需要时间，财商提升是终身课题
