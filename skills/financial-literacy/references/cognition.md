# 财富认知

> 来源：《财商训练工具手册》第一层

## 1. 欲望管理

### 核心概念
欲望管理是财商的基石。不是消灭欲望，而是学会区分**需求（Need）**和**欲望（Want）**。

### 关键原则

**需求的特征：**
- 维持基本生活所必需
- 没有会严重影响生活质量或健康
- 替代性低

**欲望的特征：**
- 提升生活品质而非维持
- 没有也不会有严重后果
- 常被营销和社会比较放大

### 实用工具：「冷静期法则」

在非必需消费前设置冷静期：
- **小额消费（<500元）：** 等待24小时
- **中额消费（500-5000元）：** 等待1周
- **大额消费（>5000元）：** 等待1个月

### 判断清单
购买前问自己5个问题：
1. 没有它会怎样？
2. 有没有更便宜的替代方案？
3. 我用它的频率会多高？
4. 一个月后我还会觉得它值得吗？
5. 我是在为"功能"付费还是为"感觉"付费？

## 2. 稀缺心态

### 核心概念
**稀缺心态（Scarcity Mindset）** 指当人感知到资源（尤其是金钱）不足时，大脑会自动聚焦于当下的稀缺问题，导致：
- 短视决策（只看眼前、忽略长期）
- 认知带宽被挤占（管窥效应 / Tunneling）
- 对未来缺乏规划
- 容易陷入"借新还旧"的恶性循环

### 识别稀缺心态的信号
- 总觉得"没钱所以不能..."
- 每次做决策都极度焦虑
- 只关注价格，忽略价值
- 不敢为未来投资（学习、健康、人脉）
- 陷入穷人思维的循环：缺钱→短视→更缺钱

### 如何建立富足心态
1. **量化恐惧**：把"不确定的焦虑"变成"具体的数字"（比如"如果失业我能撑3个月"比"万一失业怎么办"更能减少焦虑）
2. **先存后花**：收入到手先储蓄，而非花剩再存
3. **小额积累**：从极小的储蓄开始，建立"我有能力攒钱"的自我效能感
4. **关注增量**：与其盯着"我缺多少"，不如关注"我每月能多攒多少"

## 3. 消费心理学

### 核心概念
消费决策中常见的心理陷阱：

**锚定效应（Anchoring）**
- 看到原价999、现价399，就觉得"赚了600"
- 实际上399才是你需要评估的价格

**稀缺效应（Scarcity）**
- "仅剩3件""限时特惠"制造紧迫感
- 让你觉得不买就亏了

**社会认同（Social Proof）**
- "大家都在买""月销10万+"
- 用群体的选择替代自己的判断

**禀赋效应（Endowment Effect）**
- 一旦拥有了，就觉得它更值钱
- 导致"舍不得卖""沉没成本"心理

**支付痛感弱化**
- 信用卡、花呗、扫码支付比现金支付"痛感"更低
- 导致更容易超支

### 反制策略
| 陷阱 | 反制 |
|------|------|
| 锚定效应 | 只看最终价格，忽略原价和折扣 |
| 稀缺效应 | 问自己：如果明天不搞活动，我还会买吗？ |
| 社会认同 | 它适合我吗？别人买和我有什么关系？ |
| 禀赋效应 | 假设自己没有它，还愿意花这个价买吗？ |
| 支付痛感弱化 | 恢复现金思维：用月收入小时数衡量价格 |

## 4. 价值认知

### 核心概念
**价格 ≠ 价值。** 财商的核心在于理解两者的区别。

- **价格**：你为获取某物付出的金钱（市场决定）
- **价值**：它为你带来的长期效用（因人而异）

### 判断资产还是负债（基于罗伯特·清崎的定义）
- **资产**：能把钱放进你口袋的东西（股票收息、房产收租、知识变现）
- **负债**：把钱从你口袋拿走的东西（贷款买车、持续付费的订阅、贬值消费品）

### 金钱的时间价值
今天的100元 > 明年的100元，因为：
- 今天的100元可以投资产生收益
- 通货膨胀会让明年的100元购买力下降

### 价值思维框架
面对一笔消费/投资，用「三问法」评估：
1. **现在值多少钱？** → 市场价格
2. **未来值多少钱？** → 保值/增值/贬值趋势
3. **它对我的价值是多少？** → 对我生活的实际效用
