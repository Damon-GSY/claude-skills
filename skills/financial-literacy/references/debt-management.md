# 债务管理

> 来源：《财商训练工具手册》第六层

## 1. 好债与坏债

### 核心概念
**不是所有债务都是坏事。** 关键在于这笔债是在帮你创造价值，还是在消耗你的未来。

### 好债（生产性债务）
借钱去做能产生回报的事：
- **房贷**：虽然是债务，但房产可能增值，且替代了租房支出
- **教育贷款**：投资自己，提升未来赚钱能力
- **经营贷款**：用于产生收入的业务
- **低息融资**：用于高回报投资（需谨慎评估）

**好债的特征：**
- 利率低于你的投资回报率
- 用于购买增值资产或提升赚钱能力
- 有清晰的还款计划和还款能力

### 坏债（消费性债务）
借钱去消费不增值的东西：
- **信用卡分期**（购买消费品）
- **花呗/白条**（日常消费透支）
- **消费贷**（买手机、旅游）
- **高利贷/网贷**

**坏债的特征：**
- 利率高（通常12-24%甚至更高）
- 用于购买贬值物品
- 没有产生额外收入的能力
- 利滚利可能失控

### 债务评估框架
借债前问自己：
1. 这笔钱会为我产生回报吗？
2. 利率是多少？我能找到更低成本的替代吗？
3. 如果收入中断，我能继续还款吗？
4. 还款后我的现金流还有余裕吗？

### 还债策略
**雪球法（Debt Snowball）**
- 按金额从小到大排列
- 先还清最小的债，获得心理激励
- 适合需要成就感维持动力的人

**雪崩法（Debt Avalanche）**
- 按利率从高到低排列
- 先还利率最高的债，数学上最省利息
- 适合理性决策者

## 2. 信用管理

### 核心概念
**信用记录**是你在金融世界的"第二张身份证"。好的信用能帮你获得更低的贷款利率、更高的额度、更多机会。

### 什么影响信用
**正面因素：**
- 按时还款（最重要的因素）
- 信用历史长度越长越好
- 适度使用信用额度（使用率<30%）
- 信用类型多样化（房贷+信用卡比只有信用卡好）

**负面因素：**
- 逾期还款（影响持续数年）
- 过度申请贷款/信用卡（频繁查询记录）
- 高额度使用率（长期接近额度上限）
- 担保他人贷款（对方违约也会影响你）

### 信用卡使用原则
1. **按时全额还款**：这是最重要的一条。分期和最低还款会损害信用
2. **使用率控制在30%以内**：额度1万的卡，每月消费不要超过3000
3. **不要频繁申卡**：每次申请都会留下查询记录
4. **善用免息期**：利用账单日和还款日之间的免息期
5. **不要套现**：违法且有记录风险

### 信用修复
- 逾期记录通常保留5年，之后自动消除
- 如果有误报，可以向征信机构提出异议
- 最好的修复方式是：从现在起保持良好记录，让时间冲淡过去
