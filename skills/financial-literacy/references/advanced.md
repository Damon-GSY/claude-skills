# 高级财商

> 来源：《财商训练工具手册》第八层

## 1. 被动收入构建

### 核心概念
**被动收入**是不需要持续投入时间精力就能获得的收入。它是通往财务自由的关键。

### 被动收入类型

**资产型被动收入**
- 投资收益（股息、利息、基金分红）
- 房产租金
- 知识产权收入（版税、专利）

**系统型被动收入**
- 自建系统产生的收入（自动化业务、SaaS产品、课程）
- 需要前期大量投入，但建成后可以自动运转

**劳动杠杆型被动收入**
- 将一次劳动的成果反复出售（课程、电子书、付费内容）
- 投入一次，可以无限复制

### 构建路径

**第一阶段：夯实基础**
- 建立稳定的主动收入
- 攒够应急基金
- 开始定投指数基金

**第二阶段：多元投资**
- 指数基金为核心
- 适当增加股息股票
- 学习不动产投资知识

**第三阶段：创造系统**
- 将专业知识产品化（课程、咨询、内容）
- 建立自动化收入来源
- 利用AI等工具放大个人产能

### 注意事项
- 没有真正的"躺赚"——所有被动收入都需要前期的主动投入
- 不要为了追求被动收入而忽略主业
- 被动收入的构建是一个渐进过程，不是一蹴而就的

## 2. 财富传承

### 核心概念
**财富传承**不仅是传钱，更是传价值观和理财能力。

### 通胀：财富的隐形小偷
- 年通胀率3%：24年后，100万的购买力只剩50万
- 2024年的1万 vs 2004年的1万，购买力差距巨大
- 所以"存钱"不等于"保值"，必须投资

### 资产保值策略
1. **分散配置**：不要只持有现金，要有增值资产（股票、房产）
2. **抗通胀资产**：优质股票、核心地段房产、部分黄金
3. **不追求暴利**：长期稳定收益 > 短期暴利 + 亏损
4. **定期检视**：每半年/一年检查一次资产配置

### 传承方式
| 方式 | 特点 | 适用 |
|------|------|------|
| 直接赠与 | 简单直接 | 小额、无争议 |
| 遗嘱 | 明确意愿 | 有具体分配要求 |
| 保险指定受益人 | 免遗产税争议 | 人寿保险 |
| 家族信托 | 专业管理、灵活分配 | 大额资产 |
| 教育基金 | 投资子女未来 | 有子女的家庭 |

### 财商教育传承
最值得传承的"财富"是财商能力：
- 从小培养孩子的金钱意识
- 让孩子参与家庭财务讨论
- 以身作则展示健康的财务行为
- 给孩子犯错和学习的空间（小额零花钱管理）
