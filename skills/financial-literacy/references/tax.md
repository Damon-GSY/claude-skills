# 税务意识

> 来源：《财商训练工具手册》第七层

## 1. 税收基础

### 核心概念
了解税制不是要逃税，而是要知道你的钱实际去哪了，才能做好规划。

### 中国个税基本框架
**综合所得（合并计税）**
- 工资薪金
- 劳务报酬
- 稿酬
- 特许权使用费

**税率表（年度，简版）**

| 应纳税所得额 | 税率 |
|------------|------|
| ≤36,000 | 3% |
| 36,001-144,000 | 10% |
| 144,001-300,000 | 20% |
| 300,001-420,000 | 25% |
| 420,001-660,000 | 30% |
| 660,001-960,000 | 35% |
| >960,000 | 45% |

### 关键概念
- **起征点**：每年6万元（每月5000元）免税
- **专项扣除**：五险一金（个人缴纳部分）
- **专项附加扣除**：子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人、婴幼儿照护

## 2. 合规节税

### 核心概念
**节税 = 在法律允许的框架内减少税负。** 注意：节税合法，逃税违法。

### 主要节税途径

**充分利用专项附加扣除**
| 扣除项 | 每月额度 | 条件 |
|--------|---------|------|
| 子女教育 | 2000元/每个子女 | 有3岁-博士阶段的子女 |
| 继续教育 | 400元（学历）/3600元（证书） | 在读或取得证书 |
| 大病医疗 | 最高8万/年 | 自付超1.5万部分 |
| 住房贷款利息 | 1000元 | 首套房贷 |
| 住房租金 | 800-1500元 | 无房且租房 |
| 赡养老人 | 3000元 | 父母60岁以上 |
| 婴幼儿照护 | 2000元/每个婴幼儿 | 3岁以下 |

**年终奖优惠政策**
- 年终奖可以单独计税，也可以合并到综合所得计税
- 两者的税负不同，需要测算哪个更划算
- 年终奖有"临界点"效应（多1块钱可能多交很多税）

**公积金**
- 公积金缴存部分免税
- 提高公积金缴存比例可以减少应税收入
- 公积金贷款利率低于商业贷款

**投资税务优惠**
- 个人养老金账户：每年最高12000元税前扣除
- 国债利息收入免税
- 部分基金分红可能享受税收优惠

### 节税原则
1. **合规第一**：所有策略必须在法律允许的范围内
2. **优先用足扣除**：很多人忘记填报专项附加扣除
3. **年终奖单独计税的选择**：每年做一次测算
4. **不要为了节税而做不合理的决策**：节税的收益不应该低于因决策变化带来的损失
