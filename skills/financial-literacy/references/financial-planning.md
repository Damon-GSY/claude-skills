# 财务规划

> 来源：《财商训练工具手册》第四层

## 1. 财务目标设定

### 核心概念
模糊的目标无法执行。财务目标必须**具体、可量化、有时限**。

### SMART目标法
| 维度 | 要求 | 反例 | 正例 |
|------|------|------|------|
| Specific 具体的 | 明确描述 | "我要攒钱" | "我要攒够10万应急基金" |
| Measurable 可量化 | 有数字指标 | "多存点" | "每月存3000元" |
| Achievable 可达成 | 基于现实 | "1年存100万（月薪1万）" | "1年存3.6万" |
| Relevant 有意义的 | 与人生目标关联 | "为了攒钱而攒钱" | "攒够首付，在城市安家" |
| Time-bound 有时限 | 设定截止日 | "以后再说" | "2026年12月31日前" |

### 目标分层体系
**短期目标（< 1年）**
- 建立记账习惯
- 存下第一个1万
- 还清信用卡欠款

**中期目标（1-5年）**
- 攒够应急基金（6个月支出）
- 攒够首付
- 还清消费贷款

**长期目标（5-20年）**
- 子女教育基金
- 购房/换房
- 退休基金
- 财务自由

### 目标追踪
- 每月检查进度
- 用可视化工具（表格、App、进度条）
- 未达标时分析原因，调整方案而非放弃目标

## 2. 现金流管理

### 核心概念
**现金流 = 收入 - 支出**。正现金流让你积累财富，负现金流让你积累债务。

### 现金流三要素
1. **流入（Income）**
   - 主动收入：工资、奖金、兼职
   - 被动收入：投资收益、租金、版税
   - 增加流入的方法：加薪、副业、投资

2. **流出（Expenses）**
   - 固定支出：房租、保险、贷款
   - 弹性支出：餐饮、交通、日用
   - 可变支出：娱乐、购物

3. **净流（Net Cash Flow）**
   - 正值 = 你在积累财富
   - 负值 = 你在消耗财富
   - 目标：净流 ≥ 收入的20%

### 现金流优化策略
**开源（增加流入）**
- 提升职场竞争力 → 争取加薪
- 发展副业/兼职 → 增加收入源
- 投资理财 → 被动收入

**节流（减少流出）**
- 消除不必要的订阅和会员
- 优化大额固定支出（如换更便宜的租房）
- 减少冲动消费

**提速（提高净流率）**
- 储蓄率从20%逐步提高到30%+
- 把加薪的部分全部存起来（先储蓄后消费）

## 3. 应急基金

### 核心概念
**应急基金**是你专门留出来应对突发状况的钱——失业、生病、意外维修、家庭变故。它不是投资，是**安全垫**。

### 需要多少？
建议储备 **3-6个月的必要支出**：
- 稳定工作（体制内/大厂）：3个月
- 一般稳定：6个月
- 自由职业/收入不稳定：9-12个月

### 放在哪里？
| 存放方式 | 优点 | 缺点 |
|---------|------|------|
| 货币基金（余额宝等） | 随取随用、稳定 | 收益略低 |
| 银行活期 | 最灵活 | 收益最低 |
| 短期定期/大额存单 | 稍高收益 | 灵活性差一些 |
| **不要**放股市 | — | 可能亏损，失去安全保障意义 |

### 原则
1. **这笔钱不是用来投资的**——流动性优先于收益
2. **专款专用**——只有真正的紧急情况才能动用
3. **用后要补回来**——如果动用了应急基金，要在接下来几个月补回去
4. **没有应急基金之前，不要急着投资**——先防守，后进攻

## 4. 财务自由路径

### 核心概念
**财务自由**不是要有很多钱，而是你的**被动收入能够覆盖日常支出**。

### 财务自由公式
被动收入 ≥ 日常支出

### FIRE 简化计算
**4%法则（Trinity Study）**
- 如果你每年需要支出X元
- 你需要积累 25 × X 元的投资资产
- 按4%的提取率，这笔钱理论上可以永远花不完

举例：
- 每年支出12万 → 需要300万
- 每年支出20万 → 需要500万

### 财务自由阶段
1. **依赖期**：完全靠工资
2. **蓄力期**：工资为主，开始有少量被动收入
3. **过渡期**：被动收入覆盖50%+支出
4. **自由期**：被动收入完全覆盖支出

### 达成路径
- 提高储蓄率（这是你能控制的最大变量）
- 学习投资理财（让钱生钱）
- 发展多元收入（不要只依赖一份工资）
- 降低不必要的支出（减少"自由"所需的金额）
- 保持耐心（这是一个以十年为单位的工程）

### 注意
- 财务自由不是终点，而是一种**选择权**
- 不一定要完全"不工作"——而是可以只做想做的事
- 过度节俭追求提前退休可能损害生活质量和健康
