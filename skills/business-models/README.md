# Business Models Skill

商业思维模型顾问 Skill，基于《经典商业思维模型手册》25个经典商业思维模型。

## 安装

```bash
clawhub install business-models
```

或手动安装：
```bash
git clone https://github.com/Damon-GSY/business-models-skill.git
cp -r business-models-skill ~/.openclaw/skills/
```

## 模型概览

共 25 个模型，分八层结构：

| 层级 | 数量 | 模型 |
|------|------|------|
| 底层认知 | 3 | 万一模型、商业洞察模型、商业简史模型 |
| 商业模式 | 3 | 商业模式三要素、企业价值模型、价值空间模型 |
| 战略选择 | 3 | 十大战略模型、选择战略的战略、商业进化模型 |
| 创新与增长 | 3 | 创新利润模型、技术采用生命周期、产品生命周期 |
| 产品与用户 | 2 | 用户需求模型、企业能量模型 |
| 营销与渠道 | 3 | 新零售模型、购买路径模型、私域模型 |
| 管理与组织 | 4 | 管理的七种兵器、薪酬五个抓手、企业生命周期、三个共同体 |
| 宏观经济与个人 | 3 | 经济周期模型、人生商业模式、高效能人士七个习惯 |
