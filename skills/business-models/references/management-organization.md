# 管理与组织模型

## 管理的七种兵器

企业管理框架的七个核心维度：

| 兵器 | 核心问题 | 关键要点 |
|------|---------|---------|
| **职业天梯** | 员工往哪升？ | 管理路径 + 技术路径双通道 |
| **年度考核** | 怎么考核？ | 不只看结果，还要看"怎么做到的" |
| **全面激励** | 怎么激励？ | 物质 + 精神 + 成长三重激励 |
| **技能字典** | 需要什么能力？ | 岗位能力标准化 |
| **职位设计** | 怎么设岗？ | 权责清晰，分工合理 |
| **招聘流程** | 怎么招人？ | 标准化、可复制 |
| **权力管理** | 权力如何分配？ | 集权与分权的平衡 |

**使用方法：**
搭建管理体系时，逐个检查这七个维度是否完善。

## 薪酬制度的五个抓手

| 抓手 | 说明 |
|------|------|
| **年薪总额** | 企业总人力成本预算 |
| **薪酬结构** | 工资:奖金:年终奖的比例 |
| **指标** | 考核什么，就得到什么 |
| **年终奖** | 与年度考核挂钩 |
| **年度考核** | 考核"怎么做到的"而非仅看结果 |

### 三类岗位的薪酬结构差异

| 岗位类型 | 特征 | 典型比例（工资:奖金:年终奖） |
|---------|------|--------------------------|
| **S岗**（业绩责任岗） | 销售、运营、商务 | 6:3:1（衡量责任、业绩、行为） |
| **T岗**（专业能力岗） | 技术、研发 | 8:1:1 或 8:0:2（衡量工作成果） |
| **O岗**（支持服务岗） | 财务、行政、人事、法务 | 9:0:1（高薪养廉，安全感） |

## 企业生命周期模型

三个时期，每个时期有不同的核心矛盾和应对策略：

| 时期 | 核心矛盾 | 关键人物 | 策略 |
|------|---------|---------|------|
| **创业期** | 活下来 | 雷军：不要用战术的勤奋掩盖战略的懒惰 | **用战略对冲运气** |
| **成熟期** | 管理升级 | 孙振耀：中国民营企业强化职业化管理能力的时机到了 | **从"没有管理就是最好的管理"到"向管理要效益"** |
| **转型期** | 第二曲线 | 任正非：公司为什么缺少系统性思维？ | **忘记、借用、学习** |

**转型期方法论三步：**
1. **忘记** — 过去的成功经验可能是现在的障碍
2. **借用** — 从外部引进新能力
3. **学习** — 在实践中建立新能力

## 三个共同体

企业与员工关系的三种层次：

| 共同体 | 本质 | 员工角色 | 风险/收益 |
|--------|------|---------|----------|
| **利益共同体** | 买卖关系 | 能力的出卖者 | 低风险/低收益（固定薪酬+奖金） |
| **事业共同体** | 合伙关系 | 创新冒险者 | 高风险/高收益（创新成功分享巨额收益） |
| **命运共同体** | 股权关系 | 共担风险的股东 | 最高风险/最高收益（投资入股，共担共享） |

**关键洞察：**
- 利益共同体留不住人才——能力用完就走
- 事业共同体绑定创新者——分享风险和收益
- 命运共同体深度绑定——股权连接，长期共存
- 不同岗位、不同阶段需要不同的共同体形式
