# 战略与创新模型

## 十大战略模型

战略理论分为十大学派，各有适用场景：

| 学派 | 核心思想 | 适用场景 |
|------|---------|---------|
| 设计学派 | 战略是设计出来的 | 创业初期，需要整体构想 |
| 计划学派 | 战略可以程序化 | 成熟企业，需要可执行计划 |
| 定位学派 | 战略是找到正确位置 | 竞争激烈的市场 |
| 企业家学派 | 领导者构筑愿景 | 强创始人驱动的企业 |
| 认知学派 | 战略是心智模式 | 需要转变思维方式 |
| 学习学派 | 战略在行动中涌现 | 不确定环境 |
| 权力学派 | 战略是协商的结果 | 多元利益相关者 |
| 文化学派 | 战略是企业文化 | 需要内部凝聚 |
| 环境学派 | 适者生存 | 剧烈变化的外部环境 |
| 结构学派 | 重在变革 | 转型期 |

**关键洞察：**
> 没有最好的战略，只有适合的战略。战略是药，是药三分毒。对症下药，药到病除。

## 选择战略的战略

基于三个维度选择战略姿态：

| 维度 | 含义 |
|------|------|
| **不可预测性** | 环境变化有多快？ |
| **可塑性** | 你能改变环境吗？ |
| **环境严苛性** | 竞争有多激烈？ |

五种战略姿态：

| 姿态 | 核心逻辑 | 关键动作 |
|------|---------|---------|
| **求快** | 快鱼吃慢鱼 | 行动快 + 效率高 |
| **做大** | 规模优势 | 扩大规模、建立壁垒 |
| **抢先** | 占据生态位 | 新技术/新模式/跨界应用 |
| **协调** | 利益对齐 | 平衡多方利益 |
| **求存** | 活下去 | 现金流优先，缩减开支 |

## 商业进化模型

商业进化在两条腿上交替前进：
- **创造价值** — 生产、研发
- **传递价值** — 销售、物流、营销

商业模式演进：C2B → B2B → D2b → C2C → O2O ……

**关键洞察：**
- 虚拟经济（金融）是商业世界的润滑剂、黏合剂、放大器
- 缺的不是机会，而是耐心、勇气和洞察本质的能力

## 创新利润模型

五种创新，从易到难：

| 类型 | 降低的是什么 | 案例 |
|------|-------------|------|
| **效率创新** | 生产成本 | 规模化、自动化 |
| **模式创新** | 交易成本 | 七种交易成本中降低任何一种 |
| **产品创新** | 用户成本 | 帮用户省钱/省时间，实际卖得更"便宜" |
| **系统创新** | 用同种能力满足不同需求 | 跨界应用现有能力 |
| **颠覆式创新** | 同种需求被不同能力满足 | 新技术替代旧技术 |

**关键洞察：**
- 只有创新才能带来利润，否则只是"套利"
- 每一种创新都是巨大的机会

## 技术采用生命周期

五个阶段：

| 阶段 | 占比 | 特征 | 策略 |
|------|------|------|------|
| **创新者** | 2.5% | 追求新技术本身 | 播种 |
| **早期采用者** | 13.5% | 愿意冒险，有远见 | 找到标杆客户 |
| **早期大众** | 34% | 实用主义，看口碑 | 跨越鸿沟（关键！） |
| **晚期大众** | 34% | 保守，等别人验证 | 规模化 |
| **落伍者** | 16% | 怀疑新技术 | — |

**关键洞察：**
- 创新者和早期大众之间存在"鸿沟"——这是最危险的阶段
- 炒作周期（技术成熟度曲线）帮助判断技术是"泡沫"还是"红利"
- 对AI：目前大多数AI应用处于从早期采用者向早期大众跨越的阶段

## 产品生命周期

四个阶段，每个阶段关键指标不同：

| 阶段 | 最重要的事 | 应该打造的护城河 |
|------|-----------|----------------|
| **投入期** | 创新能力 | 专利护城河 |
| **成长期** | 销售 | 规模效应护城河 |
| **成熟期** | 利润 | 网络效应 + 品牌护城河 |
| **衰退期** | 成本 | 迁移成本（减缓衰退） |

**使用方法：**
判断你的产品处于哪个阶段，然后把资源投到该阶段最重要的事情上。
