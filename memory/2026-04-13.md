# 2026-04-13 决策分析：Qwen vs 高德

## 背景

Damon 在换工作，两个候选：
- **Qwen（千问）**：Agentic RL / 基座模型能力提升（长程调用、memory）→ 千问 App
- **高德**：具身智能方向

## Damon 的核心能力

- **后训练（Post-training）**：RLHF、DPO、SFT、对齐等
- 这是他的核心竞争力，决策应围绕"后训练能力在哪里价值最大化"展开

## 对话中识别的关键信息

1. Qwen 不做具身智能，方向是 Auto Research + Agentic RL
2. 高德做具身智能，但 Damon 没接触过该领域，担心试用期被裁
3. 高德的吸引力来源：二面面试官的技术视野和愿景（但经分析，这个因素权重应降低）
4. 千问的顾虑：工作强度可能影响健康（待验证）
5. 具身智能的判断：10 年后可能有潜力，但意识到在千问积累的能力（Agentic RL、memory、长程调用）将来可迁移到具身方向
6. Damon 自己在对话中得出了"先在千问做能力积累，以后可跳过去"的推论

## 使用的思维工具

| 工具 | 作用 |
|---|---|
| 黄金圈法则 | 澄清"想做什么"比"去哪家"更重要 |
| 贝叶斯推断 | 识别对高德的判断多为先验猜测，缺乏证据 |
| 信号与噪声 | 区分面试官吸引力（表层信号）vs 团队真实情况 |
| 冰山模型 | 决策被水面事件驱动，水下系统结构信息不足 |
| 反向思考 | 选错的代价：高德可迁移性风险大，千问能力不贬值 |
| 适应度景观 | Qwen 是当前高适应度峰值，迁移到具身路径畅通 |
| 系统思考 | 分析两边增强回路和调节回路 |
| 认知偏差 | 晕轮效应（面试官好感放大）、低估自身迁移能力 |
| 转移负担 | 警告在高德可能陷入补短板而非发挥长板的陷阱 |
| 核心竞争力分析 | 后训练能力在千问是核心杠杆，在高德可能被低估 |

## 最终结论

**理性分析指向 Qwen。** 六个维度中五个偏向千问，高德唯一优势（面试官技术视野）被证明是表层信号。

## 待验证事项

1. **联系千问内部人确认工作强度**（团队节奏、加班频率、人员流失情况）
2. **如强度确实大，校准个人边界**（能否管理高强度环境？）
3. **（可选）联系高德追问**：后训练定位、团队 OKR、最大瓶颈
